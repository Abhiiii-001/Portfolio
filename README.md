A premium, fully responsive personal portfolio website built with modern web technologies. This project showcases my professional journey, skills, and projects with a sleek, dark-themed aesthetic.

## 🚀 Features

- **Dynamic Navigation**: Smooth transitions between About, Resume, Portfolio, and Contact sections.
- **Functional Contact Form**: Integrated with **Formsubmit.co** and **AJAX** for seamless message delivery without page reloads.
- **Downloadable Resume**: Direct access to my latest CV with a single click.
- **Interactive Portfolio**:
  - Filter projects by category (Full Stack, Frontend, etc.).
  - Dual link support (GitHub Repo + Live Demo) with descriptive tooltips.
  - Glassmorphism hover effects and smooth image scaling.
- **Responsive Design**: Optimized for all devices from mobile to ultra-wide desktops.
- **SEO Optimized**: Semantic HTML and optimized metadata.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons**: [Ionicons](https://ionicons.com/)
- **Forms**: AJAX + Formsubmit.co
- **Design**: Dark-themed vCard aesthetic with gold accents.

## 📦 Getting Started

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Abhiiii-001/Portfolio.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd Portfolio
   ```

3. **Open `index.html`**:
   Simply open the `index.html` file in your preferred web browser.
