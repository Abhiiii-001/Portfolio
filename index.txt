About me

I am a versatile Full-Stack Developer specializing in building robust web applications using TypeScript-driven frontend frameworks (MERN, Next.js) and scalable Python backends (Django, FastAPI).

My job is to build your website so that it is functional and user-friendly but at the same time attractive. I created web design for many famous brand companies.
